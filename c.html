1．定义如下的函数
函数原型： int Position_By_Value(int d[],int n,int j)
功能说明： 计算数据d[j]在数组d中的“名次”。
参数说明： d    数组的起始地址（该数组内共有n个互不相同的数据）。
           n    数据个数（n>0）。
返回值：   数据d[j]在数组d中的名次。
说明：（1）数据d[j]在数组d中的“名次”是指，如果把d中的所有n个数据排成值从大到小的一个序列，d[j]在这个序列中的位置。最大数据的名次是1，次大数据的名次是2，最小数据的名次是n，等等。
     （2）不允许改变所有数据在数组d中原有的存储位置，即，不能通过排序处理来实现函数Position_By_Value的功能。
例：   设数组v中存储的6个数据如下
v	16	8	24	32	19	21
	0	1	2	3	4	5
        则函数调用Position_By_Value(v,6,3)的返回值是1。
	int Position_By_Value(int d[],int n,int j)
        {  int i,c;
           for(c=i=0;i<n;++i)if(j!=i&&d[j]>d[i])++c;
           return n-c;
        }


2．定义如下的函数
函数原型： int  Word_Bracketed(char s[]，char t[])
功能说明： 从字符串s中取被括在一对圆括号内的所有字符，以字符串格式存储到
           字符数组t中。设字符串s中要么没有圆括号字符，要么只有一对使用正
           确的圆括号字符（即左括号在前，右括号在后）。
参数说明： s      字符数组起始地址。
           t      存储结果的字符数组起始地址。
返回值：   存储到数组t中的字符串长度。
例如：（1） 设字符串str为：   Shanghai(2013).China
           函数调用Word_Bracketed(str，substr)完成后，函数的返回值是4。
           数组substr中的字符串是：  2013
     （2） 设字符串str为：   kswang@cps
           函数调用Word_Bracketed(str，substr)完成后，函数的返回值是0。
           数组substr中的字符串是空串。
			int  Word_Bracketed(char s[]，char t[])
        {  int j,k;
           for(j=0;s[j]!=’\0’;++j)if(s[j]==’(‘)break;

           for(k=0,j+=1;s[j]!=’)’;++j,++k)t[k]=s[j];
           t[k]=’\0’;
           return k;
        }

1．定义函数Converse。
函数原型： int Converse(int d[],int n)
功能说明： 计算数组d中数据的逆序数。
参数说明： d    数组的起始地址（该数组内共有n个互不相同的数据）。
           n    数据个数（n>1）。
返回值：   数组d中数据的逆序数。
说明： 数组d中数据的逆序数是指满足
           d[i] > d[j]        ( 0≦i<j<n )
的数据对的个数。
例如，设数组data中存储的5个数据如下
data  23  18  14  102 3
  0 1 2 3 4
          则函数调用Converse(data,5)的返回值是7。
定义函数Converse。
int Converse(int d[],int n)
{  int  j,k,c;
   for(c=j=0;j<n-1;++j)
     for(k=j+1;k<n;++k)
       if(d[j]>d[k])++c;
   return c;
}

2．定义函数DigitString。
函数原型： int  DigitString(char s[])
功能说明： 计算字符串s中出现的数字串的个数。
参数说明： s      字符数组起始地址。
返回值：   字符串s中出现的数字串的个数。

说明：  一个数字串是由连续n个（n≧1）十进制数字字符构成的。
例如：（1） 设字符串str为：   Shanghai(2008).China
           函数调用DigitString(str)完成后，函数的返回值是1。
     （2） 设字符串str为：   Kswang(021-62233333)
           函数调用DigitString(str)完成后，函数的返回值是2。
定义DigitString。
int  DigitString(char s[])
{  char *p,*q; int c;
   for(p=s;*p!=’\0’;++p){
      for(q=p;*q!=’\0’;++q)if(*q>=’0’&&*q<=’9’)break;
      if(*q==’\0’) return c;
      for(++c;*q>=’0’&&*q<=’9’;++q);
      p=q-1;
   }
   return c;
}

1．定义函数Converse。
函数原型： int  Top(int d[],int n)
功能说明： 计算数组d中“峰值”的个数。
参数说明： d    数组的起始地址（该数组内共有n个互不相同的数据）。
           n    数据个数（n>2）。
返回值：   数组d中“峰值”的个数。
说明： 数组d中的一个“峰值”数据d[i]满足：
           d[i] > d[i-1]  且  d[i] > d[i+1] 
例如，设数组data中存储的5个数据如下
data  23  38  14  102 3
  0 1 2 3 4
          则函数调用Top(data,5)的返回值是2（即38和102都是数组data中的峰值）。
定义函数Converse。
int  Top(int d[],int n)
{  int j,c;
   for(c=0,j=1;j<n-1;++j)
     if(d[j]>d[j-1])&&d[j]>d[j+1])++c;
   return c;
}




	
2．定义函数AlphaString
函数原型： int  AlphaString(char s[])
功能说明： 计算字符串s中出现的字母串的个数。
参数说明： s      字符数组起始地址。
返回值：   字符串s中出现的字母串的个数。
说明：  一个字母串是由连续n个（n≧1）英文字母（不论大小写）构成的。
例如：（1） 设字符串str为：   Shanghai(2008).China
           函数调用AlphaString(str)完成后，函数的返回值是2。
     （2） 设字符串str为：   Kswang(021-62233333)
           函数调用AlphaString(str)完成后，函数的返回值是1。
定义函数AlphaString。
int  AlphaString(char s[])
{  char *p,*q; int c;
   for(p=s;*p;++p){
      for(q=p;*q;++q)if(*q>=’A’&&*q<=’Z’||*q>=’a’&&*q<=’z’)break;
      if(*q==0) return c;
      for(++c;*q>=’A’&&*q<=’Z’||*q>=’a’&&*q<=’z’;++q);
      p=q-1;
   }
   return c;
}
